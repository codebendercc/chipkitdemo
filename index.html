Getting Started.

Hi there, and welcome to the ChipKIT Getting Started Demo page. This page will guide you through using your ChipKIT board for the first time, help you set up your computer to program your ChipKIT board, and it will also show you some of the capabilities of the boards.

We're using codebender, which not only features code-sharing, but also allows anyone to upload and debug Arduino sketches from within a web browser. Plus, it works on lower-tech machines, like Chromebooks!

Before starting, you need to install the right Browser Plugin. Click here for Chrome or here for Firefox.

Communication

Another one of the most simple programs you can write is the Hello World program. This sends data from your ChipKIT board to your computer

The code itself is pretty easy to understand. Go ahead, click Flash to load this code to your board.

After uploading, go to the embedded serial monitor below, set the baud rate to 9600 bps, and click Connect. You should start getting data from your ChipKIT board.
